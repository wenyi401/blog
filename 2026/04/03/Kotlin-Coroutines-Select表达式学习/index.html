前言

Select 表达式允许同时等待多个挂起函数，并选择第一个可用的。类似于 IO 多路复用，在 Kotlin 协程中提供了强大的并发选择能力。

一、基本概念

什么是 Select？

Select 表达式是 kotlinx.coroutines 的实验性功能，允许：

  • 同时等待多个挂起函数
  • 选择第一个可用的结果
  • 实现多路复用模式
