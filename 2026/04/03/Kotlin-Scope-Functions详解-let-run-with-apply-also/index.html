前言

Kotlin 标准库包含一组作用域函数，旨在通过在对象上下文中执行代码块来简化代码。这些函数包括：let、run、with、apply 和 also。

一、函数对比

函数 对象引用 返回值 使用场景
let it Lambda 结果 空安全调用
run this Lambda 结果 执行代码块
with this Lambda 结果 操作对象
apply this 对象本身 对象配置
also it 对象本身 附加操作
