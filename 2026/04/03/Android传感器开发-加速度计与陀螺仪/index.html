前言

Android 平台提供了多种传感器，可让您监控设备的运动。加速度计和陀螺仪是最常用的运动传感器，分别用于检测设备的加速度和角速度。

TYPE_LINEAR_ACCELERATION 线性加速度
TYPE_ROTATION_VECTOR 旋转矢量

七、最佳实践

  1. 及时注销：避免耗电
  2. 选择合适采样率：SENSOR_DELAY_NORMAL / UI / GAME / FASTEST
  3. 低通滤波：减少噪声
  4. 坐标系理解：设备坐标系 vs 世界坐标系
